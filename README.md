# The-Black-Widow
My Hyprland config. Using quickshell for most widgets. It comes with an install script.

# Dependencies
1) Eza - for better ls usage

2) Material Symbols Font - symbols for bar, battery etc.

3) quickshell - for widget system.

4) cava 

5) pavucontrol-qt 

6) playerctl 

7) geoclue 

8) brightnessctl 

9) ddcutil 

10) bc 

11) cliphist 

12) cmake 

13) curl 

14) wget 

15) ripgrep 

16) jq 

17) adw-gtk-theme 

18) fontconfig

19) Google Sans Flex
